# Ambassadors Competitions

A collection of interactive competition projects, challenges, and educational tools developed through the University of Arizona Computer Science Ambassadors program. This repository serves as a central place for hosting competition software, scoring systems, presentation materials, and interactive web applications built for student engagement events.

## Overview

This repository contains projects designed to support technical competitions, workshops, and challenge-based learning experiences. Each project is built with usability, accessibility, and student participation in mind, with a focus on creating engaging interactive experiences.

Current projects include:

### 1) Calculator
A browser-based calculator application built with:

- JavaScript
- HTML5
- CSS3

Features include:

- Responsive user interface
- Basic arithmetic operations
- Clean interactive design
- Fast lightweight frontend implementation

---

### 2) UA SpeedLab Car Tuning Challenge
An interactive educational competition platform where participants tune vehicle parameters and compete based on performance scoring.

Features include:

- Vehicle tuning simulation
- Custom scoring system
- Interactive challenge mechanics
- Web-based frontend
- Competition-ready deployment structure

Purpose:

- Encourage engineering thinking
- Teach optimization concepts
- Promote hands-on experimentation
- Create engaging competitive learning experiences
